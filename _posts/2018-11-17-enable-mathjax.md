To enable the JavaScript math display engine [MathJax](https://www.mathjax.org/) within the Jekyll theme [Minimal Mistakes](https://mmistakes.github.io/minimal-mistakes/), embed the following line

```html
<script type="text/javascript" src="http://cdn.mathjax.org/mathjax/latest/MathJax.js?config=TeX-MML-AM_CHTML"></script>
```

in the `_includes/head/custom.html`.
